# dsp-w215-hnap
Tool for reading data from D-Link DSP-W215 Home Smart Plug.
Tested with hardware version B1 and firmware version 2.20.<br>
See [wiki] (https://github.com/bikerp/dsp-w215-hnap/wiki) for more information.
